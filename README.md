Web Application Testing Project

## Project Overview

This project demonstrates manual testing of a web application using industry-standard QA practices.

## Testing Scope

- Login Functionality Testing
- Validation Testing
- Functional Testing
- Defect Reporting

## Test Scenarios Covered

1. Valid Login
2. Invalid Login
3. Empty Username
4. Empty Password
5. Invalid Credentials

## Deliverables

- Test Cases
- Bug Reports
- Test Evidence (Screenshots)

## Tools Used

- Microsoft Excel
- Web Browser
- GitHub

## Outcome

Successfully executed test cases, documented results, and maintained defect reports following QA testing practices.
